寫CSS之算數學

今天遇到一個問題

當大小 400*300 的圖

要塞進比例是 2.134：1 的框時

又想要她是垂直置中的

要怎麼做

試了幾個方法

  1. vertical-align
  2. line-height
  3. top
  4. bottom

最後無法就使用 margin-top 負數來解決

100vw是我目前圖檔的寬

100vw*3/4是原始比例下的高

100vw/2.134是我目前圖檔的高

所以原始比例的高 減掉 目前的高 再除2

將會是我圖檔移動後置中的距離

(100vw3/4 - 100vw/2.134)/2 = 100vw 0.375 - 100vw 0.2343 = 0.1397 100vw

答案就是設定
